# Personal website

This is my personal website and blog that I've built with [Astro](https://astro.build/) and deployed with [Netlify](https://www.netlify.com/).
Building this site has been an ongoing personal project to learn more about web development, have some fun with design, and reinforce good writing and documentation habits.

## Version numbering

My numbering isn't strict or serious.
It's meant to approximate how much progress I think I've made toward an end result I have in my head.
If `0.0.1` is the initial commit, then `1.0.0` is a point where I would consider the site a successful project that I could discuss on, well, the site.

To that end, I don't have a roadmap of features or prioritized issues to address in sequential "versions."
I'll add and fix things when I can. ¯\\\_(ツ)\_/¯
